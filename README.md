# Jangmin Map 프로젝트

##  프로젝트 소개
이 프로젝트는 사용자 위치 기반으로 AI 장소 추천, 장소 저장 및 관리, 게시판, 채팅 기능을 제공하는 웹 애플리케이션입니다.

## 주요 기능
*   **AI 장소 추천**: 현재 위치 주변의 가볼 만한 장소를 AI가 추천하고 상세 설명을 제공합니다.
*   **장소 저장 및 관리**: 추천받거나 직접 찾은 장소를 저장하고 목록으로 관리할 수 있습니다.
*   **지도 연동**: Kakao Map API를 활용하여 장소의 위치를 시각적으로 확인하고 길찾기 기능을 제공합니다.
*   **게시판**: 사용자 간 정보 공유를 위한 게시판 기능입니다.
*   **채팅**: 사용자 간 실시간 소통을 위한 채팅 기능입니다.
*   **로그인/로그아웃**: 사용자 인증 및 세션 관리를 제공합니다.

##  설치 및 실행 방법

### 1. 백엔드 설정 및 실행
1.  **프로젝트 클론**:
    ```bash
    git clone [프로젝트_레포지토리_URL]
    cd jangmin
    ```
2.  **의존성 설치**:
    Gradle을 사용하여 의존성을 자동으로 다운로드합니다.
    ```bash
    ./gradlew build
    ```
3.  **애플리케이션 실행**:
    ```bash
    ./gradlew bootRun
    ```
    또는 IDE(IntelliJ IDEA 등)에서 `main` 메서드를 실행합니다.
    기본적으로 `http://localhost:8090` 포트로 실행됩니다.

### 2. 프론트엔드 설정 및 실행
1.  **프론트엔드 디렉토리로 이동**:
    ```bash
    cd landmark-app
    ```
2.  **의존성 설치**:
    ```bash
    npm install
    # 또는 yarn install
    ```
3.  **애플리케이션 실행**:
    ```bash
    npm start
    # 또는 yarn start
    ```
    기본적으로 `http://localhost:3000` 포트로 실행됩니다.

## 💡 사용 방법
1.  **로그인**: 애플리케이션 접속 후 로그인 페이지에서 사용자 인증을 진행합니다.
2.  **AI 추천**: 메인 페이지에서 "AI 추천받기" 버튼을 클릭하거나, 검색창에 원하는 장소 유형을 입력하여 AI의 추천을 받습니다.
3.  **장소 저장**: 현재 위치 또는 지도상의 특정 위치를 "추천하기" 버튼을 통해 저장할 수 있습니다.
4.  **추천 목록**: "추천 목록" 버튼을 클릭하여 저장된 장소들을 확인하고, 지도에서 해당 위치를 볼 수 있습니다.
5.  **길찾기/상세검색**: 저장된 장소 정보 카드에서 "길찾기 시작" 또는 "상세검색" 버튼을 통해 Kakao Map과 연동된 기능을 사용할 수 있습니다.
6.  **게시판/채팅**: 상단 내비게이션 바를 통해 게시판 및 채팅방으로 이동하여 다른 사용자들과 소통할 수 있습니다.
